# 🐸 PepeOSINT ULTIMATE

**PepeOSINT** — это мощный инструмент для OSINT-разведки по открытым источникам, упакованный в удобный Telegram-бот с фирменным стилем Пепе.

## 🚀 Основные возможности
* **Телефон:** Поиск оператора, региона, проверка на спам и утечки (через LeakCheck).
* **Email:** Поиск аватара через Gravatar и проверка в базах утечек.
* **ФИО:** Быстрые ссылки на поиск в соцсетях и генерация вероятных номеров.
* **Транспорт:** Декодинг VIN-номеров (марка, модель, год).
* **Сеть:** Геолокация по IP, WHOIS и DNS-записи для доменов.
* **Крипто:** Проверка баланса BTC и ETH кошельков.
